Slot Machine



https://github.com/pinebase/slot-machine

# Author: Scott Forsberg
# Contact: contact.scottforsberg@gmail.com

# Release Date: 2019-06-08
# Version: .2



Description

	A slot machine simulator program. Three slots will display a series of randomized shapes.
	When the shapes are all the same, you win the round.



Prerequisites

	Python 3



Installation

	Download and unzip the program files, then change working directory to them:
	
		# cd slot-machine


	Execute the program with your favorite interperetor such as idle, or from the command line:

		# python slot_machine.py
